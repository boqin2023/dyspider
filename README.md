# dyspider
douyin X-Khronos,X-Gorgon,X-Argus生成，某音API调用

1、抖音Android逆向学习，可生成X-Khronos,X-Gorgon,X-Argus加密参数。

2、可获取用户详情，视频详情，搜索等

3、进行调用前需要先生成deviceid和install_id

4、仅用于学习交流，若有侵权，请联系作者dyspider@163.com
